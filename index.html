Registro y pagos

TASKTIP opera con pagos manuales revisados por el equipo. Cada usuario debe dejar datos suficientes para identificar la tarea, evitar duplicados y recibir TASK.

Datos requeridos

Alias, Telegram y wallet de pago. La wallet es la identidad principal para pagos.

No multicuentas

Una misma wallet solo puede enviar una prueba por tarea. Duplicados pueden rechazarse.

Mínimo 10 TASK

El mínimo de retiro/pago por tarea es 10 TASK. No se publican rewards menores.

Pago hasta 24h

Los pagos manuales pueden demorar hasta 24 horas según revisión y tipo de tarea.

Hash de pago

Cuando se pague, el admin podrá guardar el hash o nota de pago para control interno.

Depósito de tareas

El anunciante deposita reward + 5 TASK de fee y pega el hash. El admin verifica antes de abrir la tarea.
